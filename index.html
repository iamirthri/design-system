معرفی دیزاین سیستم دانشجویار

دیزاین سیستم دانشجویار یک راهنمای جامع برای ایجاد رابط‌های کاربری منسجم، زیبا و کاربرپسند است. این سیستم توسط امیر طاهری، طراح و مبتکر پروژه، طراحی شده است تا اطمینان حاصل شود که تمام اجزای رابط کاربری از یک زبان بصری یکپارچه پیروی می‌کنند. هدف این سیستم افزایش کارایی توسعه‌دهندگان و بهبود تجربه کاربری برای دانشجویان است.

شروع کار

برای استفاده از دیزاین سیستم، فایل CSS را در پروژه خود وارد کنید و از کلاس‌های تعریف‌شده استفاده کنید. نمونه کد: 

<link rel="stylesheet" href="/path/to/daneshjooyar-design-system.css">
برای سؤالات بیشتر، به مستندات API مراجعه کنید.

دسترس‌پذیری

دیزاین سیستم با رعایت استانداردهای WCAG:

  • کنتراست رنگ: حداقل 4.5:1 برای متن.
  • ناوبری کیبورد: دسترسی با کلید Tab.
  • ARIA: بهبود دسترسی با ویژگی‌های ARIA.

بهترین روش‌ها

  • از رنگ‌های اصلی برای دکمه‌های مهم استفاده کنید.
  • فاصله‌گذاری یکنواخت را رعایت کنید.
  • تست دسترس‌پذیری را در هر مرحله انجام دهید.
